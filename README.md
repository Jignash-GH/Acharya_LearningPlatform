# Acharya Learning Platform

Acharya is a lightweight learning platform that analyzes a learner’s coding level through quiz-based evaluation and generates a personalized learning roadmap.

The platform helps beginners understand their current programming skill level, identify learning gaps, and follow a structured path to improve their coding knowledge.

## 🚀 Live Demo

[View Live Project](https://vibe-113.vercel.app)

## 📌 Project Overview

Many students and beginners struggle with deciding what to learn next after starting programming. Some learners know the basics, some are stuck in the middle, and some do not know whether they are ready to move to advanced topics.

Acharya solves this problem by evaluating the learner through a quiz-based assessment and then generating a personalized roadmap based on their performance.

Instead of giving the same roadmap to everyone, the platform focuses on skill-level-based learning guidance.

## 🎯 Objective

The main objective of this project is to create a simple and useful learning platform that:

- Evaluates a learner’s coding knowledge
- Identifies the learner’s current skill level
- Generates a personalized learning roadmap
- Helps beginners follow a clear learning path
- Makes programming learning more structured and less confusing

## ✨ Features

- Quiz-based coding level evaluation
- Personalized learning roadmap generation
- Beginner-friendly user interface
- Skill-level-based learning guidance
- Clean and responsive frontend design
- Supabase integration for backend/database support
- Fast development setup using Vite
- Modern UI styling with Tailwind CSS
